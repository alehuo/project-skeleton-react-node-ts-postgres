# Project-skeleton

## Tech

- TypeScript
- Frontend
  - React
  - Redux
- Backend
  - Express
- Database
  - none
- Docker-compose
  - currently from database only

## Installation

- Clone project
- Go to project root
- Install dependencies `npm install`
- Run development server `npm start`
- Open browser [http://localhost:3000/](http://localhost:3000/)
